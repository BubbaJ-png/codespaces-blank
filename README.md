A high-fidelity browser simulator inspired by the Duluth Aerial Lift Bridge.

## Quick start

1. Install:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

3. Open http://localhost:5173

## Features

- Animated lift bridge with higher-detail metalwork and lighting
- Operator customization (jacket, helmet, operator-house light)
- Operator-house camera view
- Keyboard movable character (`W/A/S/D`) with dedicated camera view
- WebXR entry support via VR button and controller support

## Build

```bash
npm run build
```
